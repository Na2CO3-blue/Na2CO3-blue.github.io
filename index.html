关于“双减”政策更好实行的建议

发表于 2023-01-03
本文字数： 1.5k 阅读时长 ≈ 1 分钟

同学们可结合学习及生活实际，围绕国家“十四五”规划战略布局和对党的“二十大”展望，尤其鼓励结合首都发展，关注经济发展、社会治理、民生保障、文化事业、生态文明、青少年成长发展等领域，自主选择题目，在充分调研基础上，形成明确的案由案据，以模拟政协提案形式提出政策建议。

提案字数以1500字以内为宜。

背景和问题

在国务院办公厅印发《关于进一步减轻义务教育阶段学生作业负担和校外培训负担的意见》中，明确提出了“双减”政策，旨在有效减轻义务教育阶段学生过重作业负担和校外培训负担。双减政策无疑是合理且必要的，但是，从我们切身经历和社会观察调研来看，国家良好的希望并没有得到有效实现，有许多细节仍有待处理和落实，在我们的长期调查中，总结出了以下问题：

  1. 学生的负担特别是教育水平不那么发达的地区并没有得到有效减轻，反而有因升学压力等因素使负担相较以前有加重的趋势。
  2. 学生负担的减轻往往会带来老师更大的责任与负担。
  3. 家长、老师群体中的“双减”相关概念并未深入人心，相对传统的教育观念仍然是主流趋势。
  4. 在互联网、移动通信高速发展的今天，许多教育公司如新东方、猿辅导发展壮大，现在双减政策下，对于相关社会教育行业的“一棒子打死”趋势，使得产业面临寒冬，大量教育行业人才面临就业危机。

原因分析

  1. 教育资源不平等的问题仍需解决，地区与地区之间，学校和学校之间的教育水平差距仍然很大。
  2. 双减使得作业减少，却是将把知识掌握得更熟练的要求转移给了老师，不少学校的课后服务仍然由学校本职老师承担，使得老师需要花更多时间与精力。
  3. 教育转型仍在持续，家长“望子成龙，望女成凤”的心理仍存在，他们往往不能安心听从国家政策倡导，而会出于对更好教育资源的追求，往往会更加严格要求孩子。在社会上，仍存在唯文凭论、唯学校背景论、唯学历层次论的现象；将成绩量化仍是评定学生、老师、学校的普遍唯一标准。从而造成中考，高考指挥棒的竞争压力向小学蔓延。许多人对于双减的疑问都在于：双减一时的减轻负担，是否是绝对公平高考负担更重的债？
  4. 相关企业已在该行业深耕，已有相当巨大数量级的教育人才与公司前赴后继投入其中，类似于“一棒子”的打压无疑会造成大量舆论压力与就业问题。

建议

  1. 引导本校教职工、退休教师主动参与课后服务，在课后服务补贴上按照合理标准给予保障，并在职称评聘、表彰奖励等方面给予倾斜，激发教师工作热情和动力。同时，引入见习教师到校参与课后延学服务等方式，见习教师的“从教经历”可作为正式入职教师行业的“加分项”。通过有偿或志愿服务的形式调动社工、志愿者、在校大学生和因双减而面临失业的优秀教育人才，共同参与课后服务工作，同时对参与课后服务的教师和相关人员制定补助标准，按月及时发放，对贫困学生课后服务费用进行减免。另外，可以加强对教师教育观念培训，让学校和教师明白，刷题并不是提升学生成绩的唯一途径，劳逸结合，德智体美劳全面发展才是更好的提升学生的方式。
  2. 建立教师编制动态统筹机制，对市域内教师，编制资源进行动态调整和统筹调剂，确保各学校师资增减基本与生源增减同步，收纳因双减而失业的优秀编外人员。
  3. 加强对双减政策以及它对学生的好处的宣传，让家长知道，让社会对全面发展对于学生个人的重要性形成共识。教育不是唯分数论，教育是一个灵魂唤醒另一个灵魂。家长可以给学生报一些兴趣班而不是布置更多的课后作业，使学生全方面多层次发展。教育部门要会同妇联等部门，也可协同基层社区，办好家长学校或网上家庭教育指导平台，指导学校建立定期家访制度，完善家长培训体系，引导家长树立正确的育儿观、成才观，努力形成减负共识，为学生成长营造良好环境。
  4. 对相关企业公司企业进行合理引导，利于利用其在教育行业多年积累的资源与经验，使其致力于为教育资源使用合理化、分配公平化、获取简易化做出贡献。
