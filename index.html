EOF

输入不说明几个 Input Block，以 EOF 为结束标志

EOF (End Of File) 为预定的保留值，通常为 -1

C 语言实现

while(scanf("%d%d", &a, &b) != EOF)
或者
while(scanf("%d%d", &a, &b) == 2)

scanf 的返回值：成功读取数据的个数
两种方法中后者可读性更好

C++ 实现

while(cin >> a >> b)

输入流读到 EOF 返回 0

《走进经济学》期中问答题个人回答

发表于 2023-01-03
本文字数： 1.5k 阅读时长 ≈ 1 分钟

谈谈你对“经济学”的认识和理解

一个家庭往往面临许多决策。例如决定每个家庭成员分别去做什么，每个家庭成员能得到什么。家庭必须考虑到每个成员的能力和愿望，从而在成员之间分配资源。
如同一个大的家庭，社会也面领着众多决策。社会必须找到某种方法来决定要做哪些工作以及谁来做这些工作。社会需要一些人去种地，一些人去搬砖，一些人去写代码。社会分配人们去做工作，那就必然要将他们带来的产品和服务进行分配。社会需要决定谁吃泡面而谁吃米其林，需要决定谁开五菱宏光而谁开布加迪威龙。
由于社会的资源是有限的，稀缺的，因此不能生产人们希望拥有的所有物品与服务。正如每个家庭成员不可能得到他想要的所有东西，社会上的每个人也注定达不到他希望的最高生活水平。于是，对社会资源管理的需求促生了经济学。
经济学是研究社会如何管理自己的稀缺资源的一门学科。在大多数社会，资源并不是由一个人配置，而是通过成千上万家庭和企业的共同选择来配置。\因此，经济学研究人们如何做出决策：做多少工作、买什么、存多少钱、如何投资。经济学还研究人们如何相互影响：买家和卖家如何共同决定该物品的价格和销售量。最后，经济学分析影响整个经济的力量和趋势：平均收入的增长、失业率、价格上升的速度。
