机器学习实验报告

发表于 2023-01-03 阅读次数：
本文字数： 2.3k 阅读时长 ≈ 2 分钟

实验目的

  • 了解机器学习、深度学习的基本概念与应用方法；
    通过实验掌握机器学习有监督学习和无监督学习预测任务的基本流程；
  • 使用框架实现简单神经网络、人脸识别的基本原理与过程

实验内容

  • 通过Python语言使用Scikit-learn库进行有监督机器学习和TensorFlow单层神经网络两种方法实现手写数字识别；
  • 使用无监督机器学习实现样本聚类分析；
  • 使用openface开源项目提供的Docker容器镜像实现人脸识别

实验步骤

预备工作，使实验能够在自己的电脑上运行

  1. 将Windows 升级为专业版，以便支持以下虚拟机功能，并用北邮提供的正版激活器激活
  2. 在Microsoft Store下载并安装wsl(Windows Subsystem for Linux)，选择Ubuntu 22.04.1 WSL版本
  3. 升级并配置wsl2版本内核，更换Linux软件源并升级软件源
  4. 在wsl中安装Jupyter Notebook，作为IDE编译环境编写Python语言
  5. 将Windows下的Microsoft Edge浏览器映射入Jupyter Notebook中实现调用，免去在wsl中单独安装浏览器
  6. 在wsl中sudo install安装Anaconda 3和Scikit-learn库

有监督学习

  1. 命令行中输入Jupyter Lab进入 Jupyter Notebook，新建Python3文件
  2. 载入Scikit-learn自带数据集手写数字识别Handwritten Digits Data Set
  3. 使用matplotlib显示数据集图片
  4. 使用numpy将图片展开成向量
  5. 使用Scikit-learn中的分类器SVM，在一半数据集上训练，另一半数据集上做出预测
  6. 使用Scikit-learn自带Metrics检查预测准确率和混淆矩

无监督学习

  1. 在Jupyter Notebook编译环境中新建Python3文件
  2. 载入Scikit-learn自带数据集手写数字识别Handwritten Digits Data Set
  3. 使用numpy将图片展开成向量
  4. 使用Scikit-learn自带Metrics检查聚类效果
  5. 使用matplotlib可视化聚类结果，使用PCA降维到2维以便平面显示

单层神经网络

  1. 在Jupyter Notebook编译环境中新建Python3文件
  2. 载入Tensorflow自带数据集手写数字识别MNIST Data
  3. 利用Tensorflow构建简单神经网络，定义损失函数和优化器
  4. 在给定的训练样本上运行以上神经网络，观察error的变化，并在待测样本上做出预测
  5. 定义准确率计算方式，检查预测准确率
  6. 查看预测结果和图片

人脸识别（此实验由于缺少源代码，在实验室电脑上进行）

  1. 运行Docker实验环境
  2. 建立人脸样本库，此处建立了Curry、James、Kobe各20张照片的样本库
  3. 运行step-1_find-faces.py获取人脸位置
  4. 运行step-2a_finding-face-landmarks.py获取脸部特征
  5. 运行step-2b_projecting-faces.py获取仿射变换后的照片
  6. 运行main.lua对仿射变换后的人脸图片提取特征编码
  7. 运行align-dlib.py进行仿射变换
  8. 运行main.lua获取128维面部特征向量表示文件
  9. 运行classifier.py train训练样本集并生成分类器
  10. 运行classifier.py infer识别被测照片

实验结果

有监督学习

  • 分类器说明和每个分类的准确率，召回率和F1-score；
  • 混淆矩阵可看到测试数据集被分类的情况；
  • 训练和测试情况正确

无监督学习

  • 数据集包含10个分类（手写数字1-10），1797个样本，特征维度为64维；
  • 可以看到3个不同kmeans初始化中心点方法的聚类器的效果，注意使用PCA-based方法；
  • 初始化中心点速度极快，因为中心点更新次数少；
  • 最后在图中可以看到PCA降维到2维的数据聚类情况

单层神经网络

  • 随着迭代的进行，神经网络在数据集上的交叉熵误差越来越小，代表正在慢慢拟合训练数据；
  • 最后在测试集上的测试准确率为91.02%；
  • 选取测试集图片，进行预测结果正确

人脸识别

  • Curry、James、Kobe测试样本的测试结果均正确，且均有0.8以上的confidence

实验总结

有监督学习

  • 修改SVM参数，如惩罚因子C，rbf核函数系数gamma值均容易造成准确率，召回率，F1-score值下降，说明默认参数较为合适
  • 将gamma值改为0.001555并将tol值即停止训练的误差值大小改为1.15能够将准确率，召回率，F1-score值从默认的0.97提升为0.98
  • 适度增大惩罚因子C能改变某些数字识别的F1-socre，既有上升又有下降，但提升不明显，C值过大后影响不大

无监督学习

  • 修改n_init值改变用不同的聚类中心初始化值运行算法的次数，增大该值显示结果的时间明显延长
  • 3个不同kmeans初始化中心点方法中，所用时间k-means++>random>PCA-based

单层神经网络

  • 增大学习迭代次数会明显延长学习时间，增加测试准确率，**少量增加如1000次即能将准确率提升至92%+**，但更大的迭代次数对准确率提升不明显，耗时增加太久
  • 在增大学习迭代次数的同时减少learning rate也可能增大准确率，学习率太大会导致更新的幅度太大，就有可能会跨过损失值的极小值，取不到参数的极优值，太小会导致模型的更新速度太慢，模型收敛慢，从而需要消耗更多的资源来保证获取到参数的极优值

人脸识别

  • 增大样本数量能有效增加最后识别的confidence值
  • 对于选用的图像，尽量选择人脸占比相对较大且自然清晰的样本，对样本准确率的提升较为明显
