大数据实验报告

发表于 2023-01-03 阅读次数：
本文字数： 2.9k 阅读时长 ≈ 3 分钟

实验目的

初步建立对大数据的认识，掌握基本应用方法。运用BDAP平台，通过工作流和对应组件完成大数据中常用的数据预处理等基本操作，并使用常用的决策树分类，支持向量机SVM分类等监督学习算法和K-means聚类，Apriori关联等无监督学习算法解决一些实际问题，基本掌握Jupyter Lab的使用以及Python语言import相关库进行问题处理的方法

  3. 分析实验结果

实验步骤

熟悉大数据实验平台

  1. 加载某段时间内某大洲新冠疫情数据集 DXYArea1.csv
  2. 使用计算生成列中concat函数合并provinceName和cityName成数据列provinceCityName
  3. 列投影展示所合并得到的数据列provinceCityNam和原始的数据列city_confirmedCount

常见的数据预处理实验

  1. 加载二十大参与人员姓名数据集TwentiethCongress.csv
  2. 使用计算生成列中substring函数切割字符串Name，获取第一个字符，得到姓氏数据列familyName
  3. 以familyName为聚集列，使用 groupBy根据姓氏进行聚集，划分成不同群体，并统计每一个姓氏出现的次数fmilyNameCount
  4. 按照familyNameCount，使用 sort 排序操作对整体数据进行排序

分类算法实验

训练模型

  1. 在私有目录新建用来存放模型的文件夹
  2. 加载训练数据train.csv
  3. 设置相关参数生成决策树模型

验证模型

  1. 加载验证数据val.csv
  2. 使用已有模型预测结果
  3. 使用多分类评估器对结果进行验证，同时修改参数，防止过拟合

测试数据

  1. 加载测试数据test.csv
  2. 使用修改参数后的模型进行预测
  3. 使用多分类评估器对accuracy进行评估

聚类算法实验

实验1

  1. 加载奥运会运动员数据集athlete_events_trunc.csv
  2. 使用填充/删除空白行中drop直接删除空白行
  3. 使用groupBy将运动员按照项目Event进行聚合，并取身高Height和体重Weight的平均值
  4. 连续两次使用计算生成列的div函数进行BMI计算
  5. 使用填充/删除空白行中fill，将空白数值填充为其余数值的平均数
  6. 使用K-均值以Event为标签列，BMI为聚类依据进行聚类

实验2

  1. 加载疫情时期各国数据集covid_19_data_trunc.csv
  2. 使用groupBy将城市粒度的确诊Confirmed，死亡Deaths，恢复Recovered数据汇总加和到国家及区域粒度
  3. 连续两次使用计算生成列的div函数计算死亡率以及治愈率
  4. 使用K-均值以Country/Region为标签列，death_rate和recovered_rate为聚类依据进行二类聚类

关联规则算法实验

  1. 加载2015年以来最受欢迎电影的演员表casts.csv
  2. 调整相关参数，使用Apriori进行关联

SVM算法实验

下载数据集

  1. 在UCI上下载鸢尾科花特征数据集iris.data
  2. 使用BDAP的在线编程模块使用Python语言，利用pandas库进行数据格式转换，利用sklearn库将数据集划分为训练集iris_train.csv和测试集iris_test.csv
  3. 下载csv文件并上传BDAP平台

训练模型

  1. 加载训练数据iris_train.csv
  2. 设置相关参数，训练生成多分类SVM模型

测试模型

  1. 加载测试数据iris_test.csv
  2. 利用训练出的模型对数据进行分类
  3. 使用多分类评估器对accuracy进行评估

手写数字预测实验

  1. 在线编程jupyter lab中import sklearn中的Digits数据集
  2. 按照1：1的比例将数据集切分为训练集和测试集
  3. 对训练集进行训练
  4. 在测试集上预测

实验总结

  • 在“熟悉大数据实验平台”实验中，熟悉了BDAP平台的文件系统，工作流搭建等基本功能，了解到工作流能对数据进行获取、预处理、探索、分析、可视化，简化实验流程与上手难度
  • 在“常见的数据预处理实验”中，掌握了一系列对数据进行处理的操作
  • 在“分类算法实验”中，初步了解了机器学习中有监督学习和无监督学习的区别，了解了“决策树”这一十分常用的有监督学习方法，即根据样本的属性特征，对输入的样本进行分类，并知道了信息增益，训练模型时选择信息增益最大的特征作为节点特征，知道了用验证集来防止过拟合的方法
  • 在“聚类算法实验”中，了解了“聚类”这一典型的无监督学习方法，聚类是对样本点进行分类，使得相似的样本分在同一类，并了解了聚类算法中经典的K-means算法，首先选取K个随机的点作为聚类中心，对于数据集中的每一个数据，按照距离𝐾个中心点的距离，将其与距离最近的中心点关联起来，与同一个中心点关联的所有点聚成一类，再计算每一个组的平均值，将该组所关联的中心点移动到平均值的位置直至中心点不再变化，当一维的聚类较为局限时，可以采用二维的聚类使聚类更加准确
  • 在“关联规则算法实验”中，认识到关联规则是反映一个事物与其它事物之间的相互依存性和关联性，如果两个或者多个事物之间存在一定的关联关系，那么其中一个事物就能够通过其他事物预测得到，还了解了项、项集、事务、支持度、可信度、频繁项集等基本概念，并学会使用Apriori算法，根据有关频繁项集的性质逐层搜索迭代，即利用k-项集探索(k+1)-项集
  • 在“SVM算法实验”中，理解了SVM算法，SVM算法是一个分类算法，基于训练集在样本空间中找到一个划分超平面，将不同类别的样本分开，知道了距离平面最近的点到平面的距离最大的平面分类效果最好，也知道了用拉格朗日乘子法求解这样的平面，同时还知道了针对一些问题的解决方法，如采取软间隔的方法来允许算法在一些样本上出错，在目标函数中加入惩罚项来让算法能够得到更好的应用，线性不可分情况使用核函数进行升维操作，在更高维度的空间中找到划分平面的同时不提高计算复杂度
  • 在“识别手写数字实验”中，学会了Python语言以及Jupyter Lab在线编程的基本使用方法，学会在Python中import使用sklearn中的SVM算法进行问题处理
