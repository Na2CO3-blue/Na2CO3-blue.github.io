EOF

输入不说明几个 Input Block，以 EOF 为结束标志

EOF (End Of File) 为预定的保留值，通常为 -1

C 语言实现

while(scanf("%d%d", &a, &b) != EOF)
或者
while(scanf("%d%d", &a, &b) == 2)

scanf 的返回值：成功读取数据的个数
两种方法中后者可读性更好

C++ 实现

while(cin >> a >> b)

输入流读到 EOF 返回 0

相关大小

64 位 long long 最大 9223372036854775807

约数个数定理

对于一个大于 1 的正整数 n 可以分解质因数：$n=\prod_{i=1}^{k}{ {p_i}^{a_i} }$
则 n 的正约数的个数就是 $f(n)=\prod_{i=1}^{k}{a_i+1}$

防爆处理

为防止爆 int 或 long long，可采用先除后乘的思考
如采用
LCM(A, B) = A / GCD(A, B) * B
而不是
LCM(A, B) = A * B / GCD(A, B)
